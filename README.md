Ao utilizar, no geral criar a node_modules com os seguintes comandos:
```
npm install
```
Ao entrar na pasta lambda, também será necessário instalar:
```
npm install
npm i mongodb
```
Utilizado:
```
> AWS Lambda - funções
> SLS Framework - SLS Deploy
> Insomnia, get e put.
> MongoDB Compass
> WSL 2
```
